Fila A

Esercizio 1
Calcolare i seguenti limiti

  1. \(\lim\limits_{x \to 4^{-}} \dfrac{3x + 1}{4x - 16}\)

  2. \(\lim\limits_{x \to 0^{+}} \dfrac{2x - 4}{1 - 3^x}\)

  3. \(\lim\limits_{x \to 2^{+}} \dfrac{x}{ln(x - 2)}\)

  4. \(\lim\limits_{x \to -\infty} (3e^{2x} - \,2e^{-2x})\)

Esercizio 2
Calcolare i seguenti limiti

  1. \(\lim\limits_{x \to -\infty} \dfrac{x^3 - 4x^5 + 1}{x^2 - 3x^4 + 2}\)

  2. \(\lim\limits_{x \to +\infty} \dfrac{1 + x^2 + \sqrt{x^3 + x^2 - 1}}{\sqrt{x^5 + 3x - 2} - x + 4}\)

  3. \(\lim\limits_{x \to 0^{+}} \dfrac{sin(x)}{1 - cos^2(x)}\)

  4. \(\lim\limits_{x \to +\infty} \sqrt{x} \, sin\left(\dfrac{1}{x}\right)\)


Fila B

Esercizio 1
Calcolare i seguenti limiti

  1. \(\lim\limits_{x \to 3^{-}} x^2\,ln(3 - x)\)

  2. \(\lim\limits_{x \to 2^{+}} \dfrac{-5x + 1}{6 - 3x}\)

  3. \(\lim\limits_{x \to 0^{+}} \dfrac{5x + 4}{2^x - 1}\)

  4. \(\lim\limits_{x \to -\infty} \dfrac{2e^{x}}{3\,e^{-x} + 7}\)


Esercizio 2
Calcolare i seguenti limiti

  1. \(\lim\limits_{x \to -\infty} \dfrac{7x^2 - x^4 + 1}{9x + 5x^3 + 2}\)

  2. \(\lim\limits_{x \to +\infty} \dfrac{\sqrt{x^7 + x - 1} + x^4 - 2}{1 - 2x^2 + \sqrt{x^5 + 3x - 2}}\)

  3. \(\lim\limits_{x \to 0^{-}} \dfrac{tan(x)}{x - x\,cos(x)}\)

  4. \(\lim\limits_{x \to +\infty} x \, sin\left(\dfrac{1}{\sqrt{x}}\right)\)
